Hardware Architecture

In order to look into the connection between software and the underlying hardware, I studied the CS61C course with a focus on the RISC-V instruction set architecture (ISA). It was great to see how a software program is translated into machine-executable code. It also lays the foundation for the study of Operating Systems Engineering course.

Operating Systems

The 6.1810 Operating Systems Engineering course by MIT let learners implement features on a minimal version of Unix v6 (xv6). With the "learning-by-doing" philosophy, it was encouraging to see simple feature being added to the operating system and performs as expected.
